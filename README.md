# Sámi Adventure - Educational Game

An open-world educational game designed to teach Sámi culture and language to 4-7 grade students in Norway.

## Features

- **Open World Exploration**: Navigate through a beautiful 2D world
- **Building System**: Build traditional Sámi structures (Lávvu tents, Reindeer farms, Storage)
- **Task System**: Complete educational tasks to learn about Sámi culture
- **Progression**: Level up, earn XP, and grow your reindeer herd
- **Save System**: Automatic saving every 30 seconds + manual save option
- **Educational Content**: Learn Sámi words and cultural information
- **Interactive Learning**: Click on buildings to learn more about them

## How to Play

### Controls
- **Arrow Keys** or **WASD**: Move your character
- **Mouse Click**: Interact with buildings or place new buildings
- **M Key**: Open/Close Menu
- **B Key**: Open/Close Building Panel
- **T Key**: Open/Close Task Panel

### Gameplay

1. **Explore**: Use arrow keys or WASD to move around the world
2. **Build**: Press B to open the building menu, select a building, then click where you want to place it
3. **Complete Tasks**: Press T to view available tasks. Complete them to earn XP and rewards
4. **Interact**: Click on buildings when you're near them to interact (collect reindeer from farms, learn about structures)
5. **Learn**: Educational popups will appear when you build structures or interact with them
6. **Progress**: Level up by earning XP, grow your reindeer herd, and unlock new content

### Buildings

- **Lávvu (Tent)**: Traditional Sámi dwelling (Cost: 50 XP)
- **Boazodoallu (Reindeer Farm)**: Raise and collect reindeer (Cost: 200 XP)
- **Gárdi (Storage)**: Store your supplies (Cost: 100 XP)

### Tasks

Complete various tasks to learn about Sámi culture:
- Build your first Lávvu
- Create storage
- Start reindeer herding
- Grow your herd

## Running the Game

Simply open `index.html` in a modern web browser. No installation required!

The game uses:
- Phaser.js 3.70.0 (loaded from CDN)
- LocalStorage for save data
- Modern HTML5/CSS3/JavaScript

## Educational Goals

This game teaches:
- Sámi vocabulary (Lávvu, Boazu, Boazodoallu, Gárdi, etc.)
- Traditional Sámi architecture and structures
- Reindeer herding culture
- Sámi way of life and traditions

## Save System

The game automatically saves every 30 seconds. You can also manually save from the menu (M key). Your progress is stored in your browser's localStorage.
